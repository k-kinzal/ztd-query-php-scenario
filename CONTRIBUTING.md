# Contributing

Thank you for taking an interest in this repository.

This repository is maintained as a public scenario factory for `ztd-query-php`, but it does **not** accept external contributions, issues, proposals, or support requests here.

## What to do instead

- For reproducible bugs or usability problems in `ztd-query-php`, open an issue at <https://github.com/k-kinzal/ztd-query-php/issues>.
- For the library source itself, use the upstream repository at <https://github.com/k-kinzal/ztd-query-php>.

## External pull requests

External pull requests opened against this repository will be closed without review so that maintenance stays focused on the scenario-factory workflow.

## Maintainer workflow

Repository maintainers use this repository to:

1. add or refine executable scenarios in PHPUnit;
2. update the EARS specification in `spec/`;
3. route validated upstream issues when a scenario reveals a real defect or serious usability problem.
